## 介绍

[极客学院](www.jiker.com)上的一个课程，**测试驱动开发实战营**。

**课程目标**

掌握测试驱动开发的方法，练好基本功、改进工作方式，提高开发效率，成为敏捷开发的第一批受益者。

**主体内容**：3 个测试驱动案例，反复实践

- FizzBuzz
- Args
- Mars Rover

## FizzBuzz

**需求**

1. 打印出从1到100的数字；
2. 将其中3的倍数替换成“Fizz”；
3. 5的倍数替换成“Buzz”；
4. 既能被3整除、又能被5整除的数则替换成“FizzBuzz”。
